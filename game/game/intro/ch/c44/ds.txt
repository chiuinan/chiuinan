名稱：賭神 (God of Gamblers，中文版)
簡介：大宇1990年出品的賭博遊戲，共提供休閒、對戰和ＲＰＧ等三種模式。
      在休閒模式中提供了樸克、押注、BAR台、水果盤、ADD 9、21點等６種
      賭博遊戲，對戰模式則以梭哈為基礎，目標在打敗１２位對手，ＲＰＧ
      模式則結合前述各種賭博遊戲，劇情描寫主角女友被山口組抓走，而主
      角必須以賭技一一擊敗對手，救出女友。由於賭博遊戲的種類相當多，
      因此頗適合無聊時消遣之用，只是操作上的設計稍嫌複雜了些。
保護：顏色密碼，破解碼：
      DS3R（彩色）或GOG1.EXE（單色）

操作：

一. 遊戲開始時的操作

上鍵 = 選擇模式
Enter = 確定模式
Esc = 結束遊戲

二. 各種賭博遊戲簡介與操作方式

1.樸克

  說明：整副牌的規格以五張牌為準，初次開牌時可選擇換牌一次，以換牌後
        的牌型組合為準決定牌的大小和所贏的倍數。同花順１００倍，四枚
        ５０倍，福祿２０倍，同花１０倍，順子５倍，三條３倍，雙對２倍，
        單對１倍，無賴則為０倍。
  操作：Enter = 開牌
        Space = 下注／確定保留牌
        左右鍵 = 選擇保留牌
        下鍵 = 離開

2.押注

  說明：檯面上共有七張牌，一張為莊家，其餘六張可押注猜那一張比莊家大，
        若所押注的牌面比莊家牌面大時，押注加倍，若牌面小時押注金額沒
        收。
  操作：1∼6 = 選擇位置並押注（１０秒內）
        下鍵 = 離開

3.BAR台

  說明：以押注的物品為準，若燈號轉至該物品上則依其倍率乘上押注金額為
        得分，賭倍則為猜中那一邊而將得分乘以２倍或沒收。荔枝２倍，蘋
        果５倍，橘子、香菇和鐘為１０倍，西瓜２０倍，雙星３０倍，雙七
        ４０倍，ＢＡＲ為５０倍。
  操作：1∼9 = 下注
        Enter = 啟動
        左鍵 = 選擇押注物／賭倍押小
        右鍵 = 選擇押注物／賭倍押大
        上鍵 = 得分
        下鍵 = 賭倍／離開

4.水果盤

  說明：顧名思義為出現均為水果時為大獎，其中７為特別獎，以轉出兩個以
        上為準，會額外增加獎額。若荔枝出現在每組第一個時也有兩倍的安
        慰獎，其餘以同樣三個連成一線時決定其獎額。賭倍時以７號為準猜
        下一張牌為大於或小於７，猜中一次可有一個紅利號碼，若將紅利號
        碼停在７或３號上則有不定數的紅利。
  操作：Enter = 啟動
        Space = 下注／停止紅利號碼
        左鍵 = 賭倍押小
        右鍵 = 賭倍押大
        上鍵 = 得分
        下鍵 = 賭倍／離開

5.ADD 9

  說明：以加到９點為最大，遊戲時有四組牌和你自己的一組牌比較，至多三
        張至少兩張取其和之個位數字判定大小。贏則賭注加倍，輸則沒收籌
        碼。（賭倍時以目前亮出的牌大小為準，猜大於它或小於它，贏則加
        倍，輸就沒有了）
  操作：Enter = 開始發牌／停止加張
        Space = 下注／加張
        左鍵 = 賭倍押小
        右鍵 = 賭倍押大
        上鍵 = 得分
        下鍵 = 賭倍／離開

6.21點（Black Jack）

  說明： 以加點數為準，其中Ｊ∼Ｋ的牌代表十點，２∼１０的牌取其大小
         為點數，Ａ則可當１點或１１點。一開始時每人有兩張牌，若正好
         一張為１０點另一張為Ａ（即１１點），則判定為ＢＪ，可獲獎金
         三倍，若小於二十一點則可叫牌，最多叫到五張，若五張牌總和未
         超過二十一點，則獲得獎金五倍，但若加到２１點以上即為爆過，
         也就是輸了，其餘待莊家開牌後比較大小，贏則加倍、輸則沒收籌
         碼。
  操作：Enter = 下注／執行
        Space = 下注
        左鍵／右鍵 = 選擇功能
        下鍵 = 離開

7.梭哈

  說明：以傳統梭哈為準，半副牌的規格（即A、K、Q、J、10、9共二十四張
        牌，其中Ａ可當１４或８點），每次對戰七回合（ＲＰＧ模式下為三
        回合）。開始時每人發三張牌，可選擇蓋任何一張，以剩餘兩張牌面
        較大者喊牌，可下注籌碼５－２５個，對手可追加或保留一次，重覆
        二次後發完各五張牌，若兩者均叫牌則開牌決勝負。牌面大小順序為：
        同花順＞四枚＞福祿＞同花＞順子＞三條＞雙對＞單對＞無賴。七回
        合結束前若有一方籌碼已成負值即判定為輸家。
  操作：1∼3 = 選擇蓋牌
        Enter = 蓋第一張／確定
        左右鍵 = 選擇選項
        1∼5 = 選擇加注金額

三. ＲＰＧ模式提示與操作

1.體力不足時要盡快到麥當勞或７－１１買漢堡可樂吃吃，或是到旅館住宿或
  休息，若體力值降到０時遊戲便結束了。
2.一開始要想先辦法賺錢及購買物品，各物品的作用如下：

    項鍊＝在適當的時機送給某人以獲得重要訊息。
    手錶＝在狀況表中會顯示目前時間，其格式為由左至右３個數字分別表示
          時、分和星期。
    股市分析＝購買後即可看到兩家股票的日線圖，藉以分析該股票明天的
          走勢。
    金幣＝用以保值，只能擁有不得賣出，但在打敗所有在外的對手後，可增
          值換回現金。
    手鎗＝擁有後可在某一適當時機發揮相當大的的用處。
    眼鏡＝可在對戰時看透對方底牌，其標示方式為Ａ一點，Ｋ兩點，Ｑ三點，
          Ｊ一直線，９和１０則沒有任何標示。

3.賺錢的方式除了休閒模式及對戰模式中介紹的所有賭博遊戲外，還可經由股
  票買賣及簽六合彩來賺錢。股票買賣的看板上有三欄數字，分別顯示前日收
  盤價、本日現價、漲跌幅度，而買賣時間為週一至週六每日上午九時至十二
  時。簽六合彩則是從６０個號碼中，簽下６個不同的號碼，每週二、四下午
  四點止簽，五點開獎，開獎後兩個小時內對獎。其獎金算法如下：
　　　特獎：六個全中，每張二佰萬
　　　頭獎：中五個，每張三十萬元
　　　貳獎：中四個，每張二萬元
　　　參獎：中三個，每張五仟元
　　　肆獎：中二個，每張二仟元
　　　伍獎：中一個，每張一仟元
  遊戲中有許多小道消息可提供股票及六合彩的情報，必須多找人談話。
4.遊戲的目標是儘量和在外的敵人以梭哈對戰，以提升經驗值（升級時主角能
  力亦同時提升）。當打敗所有在外的敵人，再進入Casino的建築物，打敗山
  口組組長便算過關了。其中有一些敵人會到處移動，可以在某個建築物門前
  站著，等遇到他後便可以選擇對戰了。

四. ＲＰＧ模式操作

1.一般操作

  上下左右鍵 = 移動
  上鍵 = 進入樸克、ＡＤＤ　９、水果盤、ＢＡＲ台等遊戲（在機台前）
  Enter = 交談／消除狀況表／進入押注、２１點等遊戲（在機台前）
  Esc = 存取檔／結束遊戲
  C = 查看主角狀況
  Q = 回到地圖初始位置
  Z = 吃巧克力補充體力（在對戰時）

  註：進入賭博機台前要先換籌碼才能玩。

2.進證券行

  左鍵 = 買入股票
  右鍵 = 賣出股票
  下鍵 = 離開
  Enter = 確定

3.進彩券行

  上下左右鍵 = 選擇號碼
  Enter = 確定／對獎

4.進購物店／７－１１商店

  上下左右鍵 = 選擇商品
  下鍵 = 離開（當移到店員時）
  Enter = 購買

5.到賭場

  Enter = 交換現金或籌碼
  下鍵 = 離開

6.到麥當勞

  上下左右鍵 = 選擇食物
  下鍵 = 離開（當指標移至最下方選項時）
  Enter = 購買
