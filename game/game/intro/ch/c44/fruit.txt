名稱：水果盤 (Fruit Bar，中文版)
簡介：智冠（軟體世界）1989年出品的９７連線水果盤遊戲，作者徐國棟，以
      第一代賭博性水果盤電玩為主要模仿對象。這個遊戲的中獎率並不高，
      純粹靠運氣，並不會像真的賭博性水果盤電玩，會有吃多少吐一定比例
      的設定，因此要累積分數並非易事。若單純憑運氣就算了，最氣的是，
      比大小時，電腦很明顯在做弊，十比九必輸。但無論勝負如何，在沒有
      金錢的賭博誘因下，９７連線水果盤遊戲其實是很無聊的，若非玩家對
      ９７連線水果盤情有獨衷，否則並不建議玩這類的遊戲。
保護：無
備註：用DoxBox玩時，必須設定：
        machine=hercules
        aspect=false
限制：排行榜輸入名字時，不可超過19個字元，否則會導致排行榜錯亂。
操作：

空白鍵 = 下注（滿72便自動旋轉轉盤）
Enter = （以上次下注金額）開始旋轉轉盤
1 = 停止第一排轉盤
2 = 停止第二排轉盤
3 = 停止第三排轉盤
Backspace = 停止所有轉盤
G = 得分／立即得分（不慢慢計數）
C = 比大小
S = 比7小
B = 比7大
Esc = 結束遊戲

說明：

比大小時，每勝一次，分數變2倍（若是7則一律算玩家勝），同時下方會出轉
動數字的巫師，出現特殊數字可額外得分：

  7 = 下注總額x2
  33或77 = 下注總額x20
  三個數字相同 = 下注總額x50
  333或777 = 下注總額x150
