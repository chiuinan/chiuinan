名稱：宇宙神風號 (Silpheed : Super Dogfighter，英文版)
簡介：Game Arts 1988出品，由Sierra公司發行的太空仿3D射擊遊戲，共有20
      關。己艦被射擊到時，先損失防禦值，只是防禦值只有６，而且少有可
      修復的機會。此外，雖有一些武器可選擇，但都不是很強力，以致要全
      部過關是不太可能的事。特別是第２０關的敵軍主艦，可說是打不死的
      （射擊了一個小時仍然不死），讓人覺得有點不爽。不過整體而言，這
      款遊戲還算是不錯的，雖然畫面醜了些，玩起來還是蠻好玩的。但最好
      能先用整人專家等遊戲修改程式鎖定防禦值，比較不會一下子便死掉。

修改：用GameMaster找防禦值（一開始是6）後

-- -- -- -- -- -- -- -- MM -- -- -- -- WW SL SR
II NL NR -- -- -- -- -- -- FF RR -- -- -- -- --

MM = 目前關卡（1∼20），修改無用
WW = 威力，1∼4（可以更大）
SL/SR = 選擇的左右翼武器，0∼4
II = 無敵剩餘時間，00=結束
NL/NR = 可選擇的左右翼武器，0∼4
FF = 01表連射，00表無
RR = 防禦盾值，最大到6

操作：

Enter = 重新開始
F1 = 音樂開關
F2 = 音效開關
F9 = 跳到前一關
F10 = 跳到最高關
Esc = 暫停

說明：

螢幕右上字母：

W = 威力，最多3級
S = 增加速度，引擎受損會失去
F = 自動連發射擊
B = 飛彈防衛系統，不怕子彈攻擊，但被撞擊會失去

其他可取得的字母：

A = ３顆護艦衛星，可抵消敵人砲火，抵消到一定程度或過關時消失
D = 畫面上敵人全毀
H = 補充防禦盾1點
I = 一段時間無敵
N = 5000分
R = 防禦盾全滿

武器（每過一關會自動增加）：

1.前方光束（Forward Beam）：剛開始使用的武器
2.方陣光束（Phalanx Beam）：攻擊範圍最廣，且可攻擊側面，只是威力太弱
3.V光束（V Beam）：前射武器，威力較強一些
4.雷射光束（Laser Cannon）：破壞力最強，但攻擊範圍最小，且可能被反射
5.自動光束（Auto Aiming）：自動瞄準敵人射擊

關卡種類：

1.太空區：各種物品或敵機雜亂出現，閃躲是主要的存活技巧
2.行星上空：沒有任何物品，必須注意不被命中
3.城堡：有各種障礙物與敵機，如何清除並躲避是重點
4.小行星區：資源最豐富的區域，須在此區補滿裝備
